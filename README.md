# Discovery-case-Siderurgia-
Case de sucesso de um projeto para realização de Discovery no setor de Siderurgia para alavancar oportunidades de melhorias nas transações da Empresa.

Case real de Discovery conduzido na área de Siderurgia, com análise de 10 áreas, mais de 50 envolvidos, 44h de imersão e 143 oportunidades mapeadas entre RPA, BI, BPMS e melhorias sistêmicas. Atuei no mapeamento AS IS, entrevistas, identificação de oportunidades, priorização e construção do roadmap de transformação.
